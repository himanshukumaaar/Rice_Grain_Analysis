## 📌 Overview
Rice Grain Analysis is a computer vision-based project designed to analyze rice grains from images. 
The system processes images to extract meaningful insights such as grain size, shape, and quality metrics.

This project demonstrates the application of image processing and machine learning techniques in agriculture for quality assessment.

---

## 🚀 Features
- Image preprocessing and enhancement  
- Grain segmentation using computer vision  
- Feature extraction (size, shape, texture)  
- Quality classification of rice grains  
- Visualization of processed results  

---

## ⚙️ Workflow
1. Input rice grain image  
2. Image preprocessing (grayscale, filtering)  
3. Segmentation of individual grains  
4. Feature extraction  
5. Classification and analysis  
6. Output visualization and metrics

## 🛠️ Setup

### Prerequisites
- Python 3.x  
- Required libraries:
  - OpenCV  
  - NumPy  
  - Matplotlib  
  - scikit-learn  

### Installation
```bash
git clone https://github.com/himanshukumaaar/Rice_Grain_Analysis.git
cd Rice_Grain_Analysis
pip install -r requirements.txt
```

### Run the Project
```bash
python main.py
```
